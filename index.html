Amazon (Unlocked Phones) Project

 

 

In this project we will use SQL to examine data. This data consists the ratings, prices, reviews and brand name of unlocked phones sold on Amazon.com.

Changes were made to the original dataset. The dataset:

Credits: https://www.kaggle.com/PromptCloudHQ/amazon-reviews-unlocked-mobile-phones 

 

 

Here, MySQL was used.

 

 

We begin by importing the dataset into MySQL. After the successful import, we can view the dataset (renamed amzn).

 

SELECT * FROM amzn;

 

We view the entire dataset here- with setup.

 

SELECT * FROM amzn ORDER BY Price;

 

 

Here, we view the dataset organised from price, ascending (LOW - HIGH). The cheapest phones appear first with the expensive phones located at the end of the table. 

We can see that OtterBox sell phones on Amazon.com at $17.95 with mixed ratings.

 

SELECT SUM(Price) AS totalvalue FROM amzn;

 

 

$278595.219999... is the total value of the phones sold on Amazon.com. The result is renamed as totavalue.

 

SELECT AVG(Price) AS averageprice FROM amzn;

 

 

The average price of a phone sold on Amazon.com is $149.94 (2dp). Again, the result is renamed as average price.

 

SELECT * FROM amzn WHERE Price > 200 AND Rating = 5;

 

 

We view the dataset where data is only shown if it meets criteria such as a price greater than $200 and a rating of 5.

 

SELECT COUNT(Price) FROM amzn GROUP BY Rating HAVING COUNT(Price) > 500;

 

 

Here, we see the number of results in amzn where a price is greater than $500.

 

SELECT Count(*) AS numofphones FROM amzn WHERE Price > 175 AND Rating = 5;

 

 

The number of phones sold on Amazon.com where price is greater than $175 and a rating of 5 is 240. The result column is renamed as numofphones.

 

SELECT * FROM amzn ORDER BY Price DESC Limit 1;

 

 

Here we get the result with the greatest price. This mobile phone by Lenovo sold on Amazon.com for $2000. This is the most expensive phone in the amzn dataset.
